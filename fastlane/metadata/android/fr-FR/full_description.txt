Confroid est une application Android permettant la gestion centralisée de configurations.
Toute application installée sur l'appareil aura la possibilité de confier le stockage de sa configuration à Confroid plutôt que de réaliser cette tâche elle-même.
L'utilisateur aura la possibilité de modifier la configuration d'une application en utilisant Confroid.
Les configurations seront stockées selon un mode transactionnel avec un historique de versions.
